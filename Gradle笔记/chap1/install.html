1. 先决条件

    2. Gradle需要JDK或JRE，版本7及以上。

    Gradle内置Groovy，不需要安装。

  2. 下载

    3. Gradle下载地址：点击这里

  3. 解压

    4. Gradle的发布是通过Zip形式，直接解压即可。

  4. 配置环境变量

    5. 配置环境变量GRADLE_HOME，指向解压路径。然后在path中添加GRADLE_HOME/bin

  5. 运行测试

    6. 输入gradle -v，显示gradle版本及其他环境变量配置。

  6. JVM选项

    7. 通常通过GRADLE_OPTS或JAVA_OPTS变量设置。
